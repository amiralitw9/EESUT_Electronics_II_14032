# Sharif University of Technology - Electronics II Project 🚀

Welcome to the official repository for the **Electronics II** course project, completed during **Term 14032 (Spring 2025)** at the **Department of Electrical Engineering, Sharif University of Technology**.

## 🎯 Project Overview
This project involves the comprehensive design, frequency analysis, and full-scale simulation of a high-efficiency **Power Amplifier** utilizing a hybrid combination of **BJT and MOS transistors** to optimize linearity, gain, and power delivery.

---

## 🛠️ Design & Simulation Specifications
* **Simulation Environment:** LTspice
* **Active Components:** BJT & MOSFET Hybrid Architecture
* **Key Analyses:** DC Biasing, AC Small-Signal, Frequency Response, and Total Harmonic Distortion (THD)
* **Documentation:** LaTeX
